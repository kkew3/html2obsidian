经过数日奋战，终于将 Zotero IF 插件开发出来。🎉

Zotero IF
一款可以为 Zotero 文献更新【影响因子】的效率工具，不过远不止于此。

An efficient tool for updating JCR IF for Zotero Items, and far more than that.

GitHub 地址

https://github.com/qnscholar/zotero-if

下载

青柠学术公众号后台回复Zotero IF获取插件；

Zotero IF 插件正式发布了！ 🎉

使用方法

前往 GitHub，了解 Zotero IF 插件的详细用法。或者前往 Zotero 设置界面，阅读 Zotero IF 插件【帮助文档】。
